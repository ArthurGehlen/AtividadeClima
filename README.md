# Atividade complementar - 02/03/2026 (Serviços em Nuvem)

Objetivo: Complemetar atividades anteriores, utilizando geocoding + cache local

## Descrição do Projeto
Este projeto consiste em um site de consulta de clima desenvolvido em ReactJS.
O usuário informa o nome da cidade e, opcionalmente, a sigla do país, e a aplicação retorna a temperatura atual em °C daquela localização.

Para obter os dados, foram utilizadas duas APIs da Open-Meteo:

- Geocoding API → responsável por converter o nome da cidade em latitude e longitude.
- Forecast API → utiliza a latitude e longitude retornadas para buscar os dados climáticos atuais.
